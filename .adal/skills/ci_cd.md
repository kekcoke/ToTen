# CI/CD & Pipeline Automation Skills

## Core Technologies & Competencies
- **Azure DevOps & GitHub Actions**: Architecting enterprise-grade delivery pipelines, utilizing YAML templates, deployment environments, and multi-stage release gates.
- **Pipeline Governance**: Leveraging the Azure DevOps REST API for large-scale automation, repository management, and environment provisioning.
- **Automated Quality Gates**: Embedding near-100% automated regression testing (Robot Framework, JMeter) directly into the CI/CD flow so quality is an integrated product discipline.
- **Artifact Management**: Setting up automated workflows to build, pack, and publish internal reusable NuGet components (e.g., `ToTen.Shared`, `ToTen.Contracts`) to Azure Artifacts or GitHub Packages.
