# UI & Frontend Quality Automation Skills

## Core Technologies & Competencies
- **Frontend Frameworks**: Experience building maintainable, testable, and performant web applications using **Angular** and cross-platform mobile apps using **React Native**.
- **UI Test Automation**: Designing robust UI automation frameworks using **Robot Framework**, **Selenium Library**, or modern alternatives like Playwright/Cypress.
- **Component & Contract Testing**: Shifting UI testing left by isolating components and mocking backend responses, alongside consumer-driven contract testing to ensure API/Frontend alignment.
- **Performance & Accessibility**: Profiling frontend performance, reducing bundle sizes, and ensuring automated WCAG accessibility compliance across long-lived platforms.
- **State Management & State Testing**: Testing complex client-side state behaviors (e.g., NgRx, Redux) to ensure predictable application flow.
