# API Development Skills

## Core Technologies & Competencies
- **C# / .NET 10**: Deep expertise in modern .NET features, high-performance API design, and maintainability for long-lived platforms.
- **Vertical Slice Architecture**: Organizing endpoints and business logic tightly by feature (e.g., `Items`, `Locations`, `Manifests`) to promote cohesion and isolated testing.
- **Asynchronous & Event-Driven Patterns**: Designing robust pub/sub mechanisms and background worker processors using MassTransit, Dapr, or native Azure Service Bus.
- **API Gateways**: Configuring edge routing via YARP (Yet Another Reverse Proxy) for rate limiting, load balancing, and cross-cutting concern aggregation.
- **API Quality Automation**: Integrating Robot Framework for API contract testing and JMeter for endpoint performance/load testing.
