# Security & Observability Skills

## Core Technologies & Competencies
- **DevSecOps (Shift-Left Security)**: Integrating SAST/DAST tools (e.g., GitHub Advanced Security, SonarQube, OWASP ZAP) natively into pull requests and deployment pipelines.
- **Threat Modeling & Governance**: Leading Architecture Review Boards (ARB), conducting continuous threat modeling, and ensuring that security and compliance are embedded by default.
- **Observability Pipelines**: Implementing robust distributed tracing, metrics, and logging through standard **OpenTelemetry (OTLP)** in .NET 10.
- **Telemetry Platforms**: Exporting telemetry to platforms like Dynatrace or Azure Application Insights to define actionable Service Level Indicators (SLIs) that demonstrate system risk reduction and performance.
